## 문자열 <-> 아스키코드
- `"A".charCodeAt(0)` = 65
- `"a".charCodeAt(0)` = 97
- `String.fromCharCode(97)` = "a"
- 알파벳 갯수는 26개
- 알파벳 map을 만드는 코드
```js
const createAlphaMap = () => {
	const map = {};

	for (let i = 0; i < 26; i++) {
		map[String.fromCharCode(i + "a".charCodeAt(0))] = 0;
	}
  
	return map;
};
```
